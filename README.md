# Fern's Sanctuary

A gentle, patient sanctuary with misty skies, unfurling seeds, and deep greens.

## 🛠️ Installation

To run this sanctuary locally for development:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/MsGuided73/fern_sanctuary.git
    cd fern_sanctuary
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Start development server:**
    ```bash
    npm run dev
    ```

## 🚀 Deployment

This project is configured for seamless deployment via **Coolify** or any **Nixpacks** compatible host.

- **Build Tool:** Nixpacks
- **Port:** 3000
- **Build Command:** `npm run build`
- **Start Command:** `npm run preview -- --port 3000 --host`

### Environment Invariants
- **Framework:** Vite + React (TS)
- **Portal:** `GardenPortal` component is required for garden interconnectivity.
